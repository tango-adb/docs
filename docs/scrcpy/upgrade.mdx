# Upgrade from 1.1.0

This page lists changes from version 2.0.0 in Scrcpy-related packages.

For changes in other packages, see [this page](../tango/upgrade.mdx).

## `@yume-chan/scrcpy`

### Add support for new options in version 3.2

- `audioSource` now includes many more options.
- `displayImePolicy` option can control whether to show IME on mirrored display.

See https://github.com/Genymobile/scrcpy/releases/tag/v3.2 for details.
