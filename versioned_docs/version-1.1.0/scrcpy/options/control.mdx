# control

:::info

Added in Scrcpy 1.22

:::

* **Type**: `boolean`
* **Default value**: `true`

Disables the [control socket](../connect-server.mdx#sockets), allowing an mirroring only mode.

## Messages

[Control messages](../control/index.mdx) are messages from client to server.

Device messages are messages from server to client, including:

* [Clipboard messages](./index.mdx#watch-device-clipboard-changes)
* UHID output messages
